📊 Matplotlib Iris Data Visualization

This project demonstrates basic data analysis and visualization using Python on the classic Iris dataset.

📁 Project Overview

The goal of this project is to explore the Iris dataset and understand relationships between features using visualizations.

We use:

Sepal length
Sepal width
Petal length
Petal width
Species (variety)

🛠️ Tools Used
Python
Pandas
Matplotlib


📊 Visualizations Included

This project includes:

Scatter plots (feature relationships)
Histograms (distribution of features)
Boxplots (statistical spread by species)
Grouped bar plots (average comparisons)

📌 Key Insights
Different Iris species show clear separation in feature distributions
Sepal and petal measurements vary significantly across species
Visual exploration helps understand data structure before machine learning


🚀 How to Run
Clone the repository:
git clone https://github.com/hasanul101/Matplotlib.git

matplotlib.ipynb
